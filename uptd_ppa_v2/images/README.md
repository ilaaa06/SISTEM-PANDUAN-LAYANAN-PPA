# Icon Assets untuk UPTD PPA

Folder ini berisi icon-icon yang digunakan di halaman beranda dan halaman admin.

## File-file yang Dibutuhkan

1. **hukum_icon.png** - Icon untuk panduan pelaporan/hukum
2. **psikologi_icon.png** - Icon untuk pendampingan psikologi
3. **dasar_hukum_icon.png** - Icon untuk dasar hukum
4. **sirine_icon.png** - Icon untuk bantuan darurat / sirine
5. **logo_ppa.png** - Logo UPTD PPA

## Spesifikasi

- Format: PNG dengan background transparan
- Ukuran Minimum: 52x52 pixels (untuk card icons)
- Ukuran Rekomendasi: 64x64 pixels atau lebih
- Style: Flat design atau modern vector
- Color: Sesuaikan dengan color scheme baru:
  - Blue utama: #36A9F0
  - Red darurat: #E93235
  - Navy: #1E3044

## Catatan

Tempatkan semua file PNG di folder ini. File akan otomatis ditampilkan di:
- Halaman beranda (index.php) - di dalam card
- Dashboard admin - sebagai icon statistic

Jika file tidak ditemukan, sistem akan fall-back menggunakan emoji icon (default).
